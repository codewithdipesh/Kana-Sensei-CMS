# Hiragana 10-Lesson Blueprint

This is the lesson structure I would use for the Hiragana basics chapter.

The goal is not just coverage. The goal is retention.

## Core Rule

Every lesson follows this learning loop:

1. `INFO` intro
2. `LISTEN`
3. `STROKE`
4. `TRACE`
5. `WRITE`
6. `QUIZ`
7. repeat for the next kana
8. end with a `REVIEW QUIZ`

## Quiz Rotation Rule

To avoid boring repetition, rotate quiz style like this:

- odd lessons: `audio -> kana`, `kana -> romaji`, `typing`, `mixed review`
- even lessons: `audio -> romaji`, `romaji -> kana`, `drawing`, `mixed review`

That keeps the learner active without making every lesson feel the same.

## Page Template For Each Kana

For each kana, use:

- `LISTEN` = hear the sound
- `STROKE` = see stroke order
- `TRACE` = trace with support
- `WRITE` = write from memory

Then place a quiz after a few kana blocks, not only at the end.

## Lesson 1 - Vowels

Covered kana:

- `あ` `a`
- `い` `i`
- `う` `u`
- `え` `e`
- `お` `o`

Page flow:

- `INFO` intro
- `a` block
- `i` block
- `u` block
- `e` block
- `o` block
- `REVIEW QUIZ`

Quizzes:

- Quiz 1: `audio -> kana`
  - Prompt: `Listen and choose the correct kana.`
  - Target: `あ`
  - Options: `あ / い / う / え`
- Quiz 2: `kana -> romaji`
  - Prompt: `What is the romaji for this kana: い?`
  - Target: `i`
  - Options: `a / i / u / e`
- Quiz 3: `romaji -> kana` typing
  - Prompt: `Type the kana for romaji "u".`
  - Target: `う`
- Quiz 4: `mixed review`
  - Prompt: `Which kana matches the sound "o"?`
  - Target: `お`
  - Options: `あ / え / お / い`

## What This Blueprint Gives You

- a consistent page structure
- sound quizzes in every lesson
- a rotated quiz style so lessons do not feel repetitive
- gradual difficulty from recognition to recall
- a final review step at the end of each lesson

## Recommended CMS Rule

For each lesson, keep this pattern:

- 1 intro page
- 3 practice pages per kana
- 4 quiz pages per lesson
- 1 final mixed review quiz

That is the best balance between teaching and memory for beginners.
